## Install [`Go`](https://golang.org/dl/)

- Linux (Ubuntu / Mint)

```command
sudo apt install golang-go
```

- Linux (Arch / Garuda)

```command
sudo pacman -S go
```

- Windows  
  https://go.dev/doc/install

```command
go install github.com/air-verse/air@latest
```
